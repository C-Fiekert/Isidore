Isidore is an open-source platform built for Cybersecurity professionals which aims to make information gathering in SOC environments easier.

Isidore allows users to query several great information resources such as Virustotal, UrlScan, and Greynoise, etc. which returns all of the information on one page in a neat and tidy fashion.

With Isidore, you can use your API keys from any of the integrated services to query information on a large scale and at a much quicker speed than the manual process.
