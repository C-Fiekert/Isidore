About the Project

I first started working on this tool in September of 2021 for my Final Year Project in college. My idea for the tool came from my experience within a SOC environment during my internship. The purpose of the tool is to be able to query multiple services at once about potentially malicious URLs, IP addresses, domains, and file hashes. The tool makes the information easy to digest and also saves time for SOC members allowing them to handle investigations quicker and more efficiently.
